#   pip install swesim[cuda]
#   This installs numba-cuda and the nvidia-*-cu12 redistributable packages.
#   CUDA_HOME is configured automatically at app startup via setup_cuda_home_from_pip().

# Option B — conda (recommended for conda environments):
#   conda install cudatoolkit
#
# Option C — system install:
#   Download CUDA Toolkit from https://developer.nvidia.com/cuda-toolkit
#   then set CUDA_HOME to the installation path, e.g.:
#   Windows: set CUDA_HOME=C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.x
